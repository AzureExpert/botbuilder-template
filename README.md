# BotBuilder Template

Sample template for building bots using Microsoft Bot Framework

## Setup

### Installing

```
npm i

```



## Deployment

### Running

```
npm run start

```

### Running tests

```
npm test
```

## Built With

* [Microsoft BotFramework](https://dev.botframework.com) - The bot framework used
